This is an eCommerce website designed using HTML, CSS, JS, Bootstrap, PHP and integrated with MySQLi database. 
Xampp server should be on, in order to run the website on your local computer.
Steps to Run the Website:
-Make sure to ennable sql and apache server on xamp on your local machine.
-Then make sure this whole ecommerce folder is in htdocs folder which is a child folder in xampp.
-Go to localhost/phpmyadmin.
-Create Database and name it store.
-In that database import store.sql which is present in project folder.
-Now just run your website.
